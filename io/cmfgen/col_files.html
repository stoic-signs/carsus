Collisional strengths¶

[2]:

from carsus.io.cmfgen import CMFGENCollisionalDataParser

Download and extract CMFGEN Atomic Data from http://kookaburra.phyast.pitt.edu/hillier/web/CMFGEN.htm

[3]:

si2_col_data = CMFGENCollisionalDataParser('./atomic/SIL/II/16sep15/si2_col')

The resulting table is stored as a DataFrame in the base attribute. Header information is saved as a dictionary in meta attribute.
